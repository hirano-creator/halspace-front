## 2. Step 1：Anthropicアカウントの作成

**重要：Claude.aiのアカウントとは別のアカウントが必要です。**

Claude Codeを使うには、`console.anthropic.com`というAPIの管理画面（Anthropic Console）にアカウントを作る必要があります。ウェブ版Claude（claude.ai）のアカウントとは別物なので注意してください。

### 手順

1. ブラウザで `https://console.anthropic.com` を開く
2. 「Sign Up」をクリック
3. メールアドレスで登録するか、GoogleアカウントやMicrosoftアカウントで登録する
4. メールに届いた確認メールのリンクをクリックする
5. 初回ログイン時に請求情報（クレジットカード）の登録を求められる

> 📸 **[画像1]** console.anthropic.com のサインアップ画面。「Sign Up」ボタンが見える状態。

> 📸 **[画像2]** 請求情報入力画面（クレジットカード番号欄）。カード番号は絶対に画像に映さないこと。

### クレジットカード登録について

「カードを登録するのが怖い」という方は少なくないと思います。ただ、**最初に課金されるわけではありません**。Anthropic Consoleは従量制で、実際に使った分だけ後日請求されます。しかも後述する「月の上限設定」をしておけば、思わぬ高額請求は防げます。

最初はまず**$5（約800円）分のクレジットをチャージ**しておくのがおすすめです。これだけで数ヶ月は十分使えます。

## 3. Step 2：APIキーの発行

Anthropic ConsoleにログインしたらAPIキーを発行します。**APIキーとは、Claude CodeがAnthropicのサーバーに接続するための「合言葉」**のようなものです。

### 手順

1. Anthropic Consoleにログイン後、左メニューの「API Keys」をクリック
2. 「Create Key」ボタンをクリック
3. キーに名前をつける（例：`my-claude-code`など何でもOK）
4. 「Create Key」を押すと、`sk-ant-api03-...` という長い文字列が表示される
5. **この文字列を必ずコピーしてメモ帳などに保存しておく**（この画面を閉じると二度と表示されない）

> 📸 **[画像3]** APIキー作成後に表示される画面。キーの文字列部分はモザイクで隠す。「Copy」ボタンが見える状態。

### ⚠️ 重要な注意

APIキーは**パスワードと同じくらい大切なもの**です。GitHubなどにアップロードしたり、人に見せたりしないでください。もし漏れたと思ったら、すぐにConsoleから削除して新しいキーを発行できます。

## 4. Step 3：Node.jsのインストール

Node.jsとは、Claude Codeが動くために必要な「土台」のようなものです。Node.js自体を使いこなせる必要はありません。ただ入れておくだけでOKです。

### 手順

1. ブラウザで `https://nodejs.org/ja` を開く
2. **「LTS版（推奨版）」**のダウンロードボタンをクリック（`v22.x.x LTS`のような表示）
3. ダウンロードされた`.msi`ファイルをダブルクリックして実行
4. インストーラーの指示に従って「Next」を押し続けてインストール完了

> 📸 **[画像4]** Node.jsのダウンロードページ。「LTS版」のボタンを赤枠で囲んで強調。

> 📸 **[画像5]** Node.jsインストーラーの画面。「Next」ボタンの位置を矢印で示す。

### インストールの確認

インストールが完了したら、正しく入ったか確認します。

1. スタートメニューを開いて「PowerShell」と検索
2. 「Windows PowerShell」を右クリックして「管理者として実行」をクリック
3. 以下のコマンドをコピーして貼り付け、Enterを押す

```powershell
node --version
```

「`v22.x.x`」のような文字が表示されれば成功です。

> 📸 **[画像6]** PowerShellでnode --versionを実行した結果。バージョン番号が表示されている画面。

## 5. Step 4：Claude Codeのインストール

いよいよClaude Code本体をインストールします。

### 手順

先ほど開いたPowerShell（管理者として実行）に、以下のコマンドをコピペしてEnterを押します。

```powershell
npm install -g @anthropic-ai/claude-code
```

インターネットのダウンロードが始まります。1〜3分ほど待つと完了します。

> 📸 **[画像7]** インストール中のPowerShell画面。文字がたくさん流れているが「これで正常」と注記。

インストールが終わったら、以下のコマンドで確認します。

```powershell
claude --version
```

バージョン番号が表示されれば成功です。

> 📸 **[画像8]** claude --versionを実行した結果。バージョン番号が表示されている画面。

### ⚠️ エラーが出た場合：実行ポリシーの変更

WindowsのPowerShellでは、セキュリティ上の理由から「外部のスクリプトを実行できない」設定になっていることがあります。以下のようなエラーが出た場合の対処法です。

**エラーメッセージ（このような文字が出たら）：**
```
このシステムではスクリプトの実行が無効になっているため〜
```

**対処法：** 以下のコマンドをPowerShellにコピペして実行します。

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

確認を求められたら「Y」を入力してEnterを押します。その後、もう一度インストールコマンドを実行してください。

## 6. Step 5：初回起動とAPIキーの設定

インストールができたら、実際に起動してみましょう。

### 起動の手順

1. まず、作業したいフォルダを作る（例：デスクトップに「my-project」という名前のフォルダ）
2. そのフォルダを**Shift+右クリック**して「PowerShellウィンドウをここで開く」を選択

> 📸 **[画像9]** フォルダを右クリックしたときのメニュー。「PowerShellウィンドウをここで開く」を赤枠で強調。

3. 開いたPowerShellに以下を入力してEnterを押す

```powershell
claude
```

4. 初回起動時はAPIキーの入力を求められます。Step 2でメモしておいたAPIキーをここに貼り付けてEnterを押します。

> 📸 **[画像10]** claudeコマンドを初めて実行したときの画面。APIキーの入力を求めるプロンプトが表示されている。

5. 利用規約への同意を求められたら「Yes」を選択します（矢印キーで選択、Enterで決定）

6. 起動に成功すると、日本語で指示を入力できる状態になります

> 📸 **[画像11]** Claude Codeが起動してプロンプト入力待ちになった画面。「>」マークが点滅している状態。

これで初期設定は完了です！

## 7. よくあるエラーと対処法

| エラーの内容 | 対処法 |
|------------|--------|
| `npm: command not found` | Node.jsのインストールに失敗している。Step 3からやり直す |
| `このシステムではスクリプトの実行が無効...` | Step 4の「実行ポリシーの変更」を実行する |
| `Invalid API Key` | APIキーをコピーし直す。先頭や末尾にスペースが入っていないか確認 |
| インストール中に止まって動かない | Enterを押してみる。それでも動かなければPowerShellを閉じて再起動して試す |
| `EACCES: permission denied` | PowerShellを「管理者として実行」で開いているか確認する |
