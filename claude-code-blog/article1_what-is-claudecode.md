## はじめに：プログラミングが「怖くなくなった」話

私はエンジニアではありません。本業は全く別の仕事で、プログラミングはあくまで「自分で使うツールを作りたい」という動機で独学しています。HTMLやPythonを少しかじった程度で、フレームワークとか設計とかは正直よくわかりません。

そんな私が、Claude Codeというツールに出会ってから、プログラミングへの向き合い方が大きく変わりました。今では小さなWebアプリや自動化ツールを自分で作れるようになっています。「私でも使えたんだから、きっとあなたにも使える」——この記事はそういう気持ちで書いています。

このシリーズでは全4回にわたって、Claude Codeの始め方から実際の使い方まで、非エンジニア目線でリアルに解説します。第1回の今回は「そもそもClaude Codeって何なの？」というところから始めます。

## 1. Claude Codeとは何か（一言で言うと）

Claude Codeとは、**AIのClaudeをパソコンの中に呼び込んで、一緒にプログラミング作業ができるツール**です。

Anthropicという会社（ChatGPTを作ったOpenAIと並ぶAI企業）が開発した、公式のCLIツールです。「CLI」という言葉が出ただけで「難しそう…」と思った方、安心してください。CLIとは「Command Line Interface」の略で、要は**黒い画面にコマンドを打ち込んで操作するもの**です。最初は少し戸惑いますが、使い方は非常にシンプルです。

## 2. ChatGPTやClaudeウェブ版との違い

「ClaudeならもうClaude.aiで使っているけど、それと何が違うの？」という疑問は当然です。私も最初そこが一番わかりませんでした。

一言で言うと、ウェブ版のClaudeは**「相談」ができるツール**、Claude Codeは**「一緒に実際に作業ができるツール」**です。

わかりやすくたとえると——

> ウェブ版ClaudeやChatGPTが「ホワイトボードの前でアドバイスをくれる先生」だとしたら、Claude Codeは「あなたのパソコンの中に直接入ってきて、一緒に手を動かしてくれる職人さん」です。

もう少し具体的に比較してみましょう。

| 機能 | Claude（ウェブ版） | Claude Code |
|------|-----------------|-------------|
| コードを提案してくれる | ✅ できる | ✅ できる |
| ファイルを直接読み込む | ❌ できない | ✅ できる |
| ファイルを直接書き換える | ❌ できない | ✅ できる |
| プログラムを実行する | ❌ できない | ✅ できる |
| エラーを自動で直す | ❌（手動でコピペが必要） | ✅ 自動で確認・修正 |
| フォルダ全体を把握する | ❌ できない | ✅ できる |

ウェブ版では「このコードを直してください」と貼り付けて、直ったコードをまたコピーして……という作業が必要でした。Claude Codeは**ファイルを直接見て、直接書き換えてくれます**。これが決定的な違いです。

## 3. 具体的に何ができるのか

では実際にどんなことをお願いできるのか、非エンジニアの私が実際に使ったことのある例を挙げます。

### ① HTMLでWebページを作る

```
シンプルなプロフィールページのHTMLを作ってください。名前と自己紹介文と連絡先が入るレイアウトで。
```

このような指示を日本語で入力するだけで、HTMLファイルを作ってくれます。ブラウザで開いてすぐ確認できます。

### ② エラーをそのまま貼り付けて解決してもらう

プログラムを動かしてエラーが出たとき、そのエラーメッセージをそのままコピーして貼り付けるだけで原因を調べて直してくれます。これが**初心者には最強の使い方**です。エラーの意味を自分で理解する必要すらありません。

```
このエラーが出ました。どうすればいいですか？

TypeError: cannot read properties of undefined (reading 'map')
    at App.js:23
```

### ③ 既存のファイルをまとめて修正する

「このHTMLのデザインを全体的にもっとシンプルにして」「このPythonコードを日本語のコメント付きにして」など、すでにあるファイルへの変更もお願いできます。

### ④ 「こんなツールを作りたい」という相談から始める

何を作ればいいかわからないときも大丈夫です。「毎日のスケジュールを入力して保存できるシンプルなWebツールを作りたい。何から始めればいい？」と相談すると、手順を考えて一緒に進めてくれます。

### ⑤ フォルダ内のファイルを横断的に修正する

複数のファイルにまたがる変更も一度で対応できます。「全部のHTMLファイルのタイトルタグの形式を統一して」といった依頼も可能です。

## 4. 料金はいくらかかるのか

Claude Codeの料金体系は**従量制（使った分だけ払う）**です。Claude.aiのサブスクリプション（月額$20程度）とは別に、APIという仕組みを使うため、Anthropicのコンソール（管理画面）でクレジットカードを登録する必要があります。

**目安の費用感（2025年時点）：**

| 使い方 | 目安の費用 |
|--------|----------|
| ちょっと試してみる（30分程度） | 数円〜十数円 |
| 1〜2時間がっつり使う | 数十円〜200円程度 |
| 毎日1〜2時間、1ヶ月使う | 1,000〜3,000円程度 |

正直、思ったより安いです。最初は「使いすぎたら高くなるのでは」と心配していましたが、個人の趣味レベルで使う分には月数百円〜1,000円台に収まることがほとんどです。

また、Anthropicのコンソールで**月の上限金額を設定できます**。「月に1,000円以上は使わない」という設定ができるので、安心して使い始められます。

## 5. 非エンジニアでも本当に使えるのか

結論から言います。**使えます。ただし、最初の2〜3日は少し戸惑います。**

私が最初にClaude Codeを使ったとき、「黒い画面」に戸惑いました。コマンドを入力して何かが動くという体験自体が初めてだったからです。でも、基本的にやることは決まっています。

1. フォルダを指定して Claude Code を起動する
2. 日本語で指示を書く
3. Claude Codeが提案してきた変更を確認して「承認」する
4. ファイルが変わっていることを確認する

これだけです。難しいコードを自分で書く必要はありません。指示の出し方にコツはありますが（第3回で詳しく説明します）、日本語で話しかけるだけで動きます。

私が最初に感じた「非エンジニアがClaude Codeを使ってよかったこと」をまとめると：

- **プログラムが動く喜びをすぐに体験できる**（最初の30分で何かが動く）
- **エラーを怖がらなくなった**（エラーが出たらClaudeに聞けばいいと思えるようになった）
- **「なぜこのコードが動くのか」を質問しながら学べる**（ChatGPTより実践的に学べる）
- **自分のペースで進められる**（締め切りも先生もいない）

一方で、正直に言うと**「使いこなせる」感覚が来るまでに1週間くらいかかりました**。最初の数回は、Claude Codeが何を言っているのかよくわからなくて、少し途方に暮れる場面もありました。でも慣れれば確実に武器になります。
