Claude Code の基本コマンド

Claude Code はターミナルから直接使えるAIコーディングアシスタント。インストールは npm 一発で完了します。

インストール

$ npm install -g @anthropic-ai/claude-code
$ claude --version
claude-code 1.0.0

使い方

プロジェクトのルートで claude を実行すれば対話的に使えます。

$ cd my-project
$ claude
> このリポジトリの構成を教えて
コードを書くより、書かせる時代へ。
