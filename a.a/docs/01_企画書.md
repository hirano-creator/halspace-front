# a.a（エー）企画書 — 経営者のための業界SNS

> 最終更新: 2026-06-21 / MVPローカル実装完了。本番デプロイ待ち。

## Context（なぜ作るか）
What'sNoは投稿（図面・現物・資料＋タグ・サムネ・Knowl/RAG）を**社内**に蓄積・共有する仕組み。その価値が社内に閉じている。a.aは、これを**社外（業界内）の経営者同士に開き**、会社・業界の情報を発信し合い、コメントで交流できるSNSにする。

公開範囲は**業界内オープン**（製造業など同業の経営者・関係者が招待制で参加する準公開コミュニティ）。`c:\dev\my-programming\a.a` は新規アプリ。バックエンドは**What'sNo API（solid-api/halspace-api）を拡張**し、認証・ストレージ(R2)・サムネ・Geminiの既存基盤を再利用する。

## コンセプト一言
**「経営者が、会社と業界の"今"を発信し合う場所」** — 補助金・セミナー・設備・図面まで、製造業の経営に効く情報が集まり、AIと人が答え、業界の資産として貯まっていくSNS。

## 主役とターゲット
- **投稿者の主役＝経営者**（中小製造業の代表・経営層）。トーンは"相談"より**発信・共有・PR**寄り。
- 現場の相談・図面Q&Aも併存（カテゴリで使い分け）。

## 投稿コンテンツ（経営者が出すもの）
- **補助金・助成金の資料/情報** — 経営者に最も実利。強いフック。
- **セミナー資料** — 学び・ノウハウの共有。
- **設備紹介・新規設備の導入動画/画像** — 自社の実力PR、引き合い創出。
- **会社のお知らせ／業界の気づき**（通常投稿）。
- **図面・現物の相談**（マスクで機密を伏せて）。
- メディア種別：テキスト / 画像 / 動画 / **資料(PDF)**。

## カテゴリ（フィードの軸）
投稿に内容カテゴリを付け、フィード上部で絞り込む。初期セット：**補助金 / セミナー / 設備紹介 / お知らせ / 相談**（メディア種別とは別軸）。

## 自動ニュース配信
フィードに**ユーザー投稿とニュースを混在**させ、投稿が無い日も開く理由を作る。
- 含めるニュース：**補助金・助成金の新着 / 製造業・業界ニュース / 制度・法改正**。
- 仕組み：artisanコマンド（`aa:import-news`）を毎時スケジュール実行。`aa_news_feeds` テーブルで管理するRSSフィードを取得 → Geminiで要約 → `aa_posts`（kind=news）に保存・重複排除（dedup_key）。
- 管理画面からRSSフィードのCRUDと手動実行が可能。ニュースは「ニュース」バッジ＋出典で投稿と区別。

## SNS連携（X / Instagram）
a.aの投稿を**後から個別にシェア**（投稿カードの「シェア」から X / Instagram へ）。投稿時の同時クロス投稿はしない（軽量・誤発信防止）。

## 「他にはない」独自性
- **軸A：図面・現物・資料という"モノ"が主役**（テキストSNSと根本的に違う）。
- **軸B：質問できるSNS（Knowl/RAG）** — 投稿が貯まるほどAIが賢くなる複利の堀（後続フェーズ）。
- **軸C：機密マスクで図面を"出せる"** — 製造業が社外に図面を出せない障壁を制度で解く。
- **軸D：補助金・業界ニュースが自動で集まる経営者コミュニティ** — 実利情報のハブ＝毎日開く理由。他SNSにない。

## 盛り上げる施策（要点）
1. 補助金など**実利ニュースの自動配信**で来訪動機を常設。
2. **What'sNoからワンクリック公開**で投稿摩擦ゼロ。
3. **シード投稿＋招待制**で信頼ある初期コミュニティから開始。
4. **設備紹介→引き合い**の導線（プロフィールの対応領域タグ）。
5. **業界特化リアクション**（👍「参考になった」）で会話の起点。
6. **週次メールダイジェスト**（注目投稿＋新着補助金）で再訪（後続フェーズ）。
7. **マスク**で安心して図面相談。
8. テーマ企画（設備ビフォーアフター等）で投稿喚起。
